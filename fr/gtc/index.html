L'Utilisateur reconnaît que l'Application dépend d'interfaces tierces hors du contrôle de l'Opérateur. L'Opérateur peut suspendre le service sans préavis si une source de données devient indisponible.

Les calculs d'"Économies nettes" sont des estimations indicatives — non une garantie financière.
